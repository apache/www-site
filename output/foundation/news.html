ASF highlights, 1999-2009

Please see our current listing of News and Announcements for more recent notes.

Keep informed about news from the Apache Software Foundation and its projects by subscribing to the Apache Announcements mailing list. To see what has happened recently, please check the archives of Apache Announcements or you can visit our foundation blog.

See the current listing of News and Announcements.

2009¶

  • November 4th, 2009 ApacheCon US 2009 Draws Unprecedented Support From the Global Open Source Community and Recognition for the 10th Anniversary and Influence of The Apache Software Foundation
    Hundreds of attendees with support from dozens of corporate sponsors, exhibitors, and community partners helped make this year's ApacheCon US among the most popular events to date. The ASF was honored by the State of California and the City of Oakland; November 4 named Apache Software Foundation Day. Read the Press Release Read More about our tenth anniversary celebration.

  • November 4th, 2009 Subversion Submitted to Become a Project at The Apache Software Foundation
    The CollabNet-sponsored Subversion project and The Apache Software Foundation (ASF) announced today that the award-winning Open Source project has formally submitted itself to the Apache Incubator in order to become part of the Foundation's efforts. Read More about this exciting development.

  • March 25, 2009
    Letter : Exactly ten years ago, the highly dedicated members of the Apache Group decided to secure the future of the Apache HTTP Server by forming The Apache Software Foundation Read more about this special anniversary in a letter by Jim Jagielski, current Chairman of the foundation.

  • March 4, 2009
    Press Release : Press Conference for the Tenth Anniversary of The Apache Software Foundation

  • March 3, 2009
    Press Release : The Apache Software Foundation Names Qpid a Top-Level Project.

2008¶

  • October 14, 2008
    Press Release : Apache Tuscany enables service oriented architecture solutions to more easily respond to new business requirements.

  • June 17, 2008
    Press Release : The Apache Software Foundation's "Apache Way" continues to gain momentum, recognition and influence on the future of open source technologies.

  • January 15, 2008
    Press Release : The Apache Software Foundation Wraps Up 2007 As Its Strongest Year To Date.

2004¶

  • September 2, 2004
    Open letter: Issued to the IETF MARID Working Group regarding the ASF's position on Sender ID.

  • May 11, 2004
    Press release: Apache HTTP Server Reaches Record Eight Consecutive Years of Technical Leadership

2002¶

  • March 26, 2002
    Press release: Apache Software Foundation Reaches Agreement With Sun Microsystems To Allow Open Source Java Implementations

2001¶

  • May 30, 2001
    Apache.org hacked but no software distributions were compromised.

    The Apache Software Foundation is officially recognized as a U.S. 501(c)3 public charity.

1999¶

  • June 30, 1999
    Press release announcing the incorporation of the Apache Software Foundation.

  • June 1, 1999
    The Apache Software Foundation is officially incorporated within the State of Delaware.
